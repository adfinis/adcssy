Table

Basic usage:

#Foo
1Bar
2Baz
 
<table class="table">
  <thead>
    <tr><th>#</th><th>Foo</th></tr>
  </thead>
  <tbody>
    <tr><td>1</th><td>Bar</td></tr>
    <tr><td>2</th><td>Baz</td></tr>
  </tbody>
</table>

Modifiers:

  • table--bordered Add border around a table
  • table--striped Add stripes to tbody rows
  • table--hover Add hover effect over tbody rows
